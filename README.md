# 🌤️ Weather Forecast

A simple Weather Forecast web application built using **HTML, CSS, and JavaScript**. It fetches real-time weather data using the OpenWeatherMap API.

## 🚀 Features

- Search weather by city name
- Displays current temperature
- Shows humidity
- Shows wind speed
- Search using button or Enter key
- Simple and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## ⚙️ How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Enter a city name to view its current weather.

## 🔑 API

This project uses the **OpenWeatherMap API** to fetch real-time weather data.
